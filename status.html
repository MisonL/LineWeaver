  • 智能处理：支持 \n、\r\n、\r 各种换行符
  • 一键复制：现代 Clipboard API + 降级方案
  • 键盘快捷键：Ctrl/Cmd+Enter 转换，Ctrl/Cmd+D 复制
  • 响应式设计：支持桌面、平板、手机全平台
  • 实时反馈：Toast 消息提示和状态显示
