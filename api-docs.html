📚 LineWeaver API 文档

纯前端RESTful接口，完全兼容GitHub Pages部署

无需后端服务器，所有API调用都在浏览器中完成

🚀 快速开始

所有API都是纯前端实现，直接在浏览器中使用：

// 使用示例 const result = await window.apiClient.convertText('示例文本', 'smart'); console.log(result.convertedText);
📋 API端点
POST /api/v1/convert 文本转换接口
将多行文本转换为单行文本，支持多种处理模式

请求参数

参数 类型 必填 说明
text string 是 需要转换的原始文本
mode string 否 处理模式：simple, smart, custom（默认：smart）
config object 否 自定义配置参数（仅在custom模式下使用）

请求示例

{ "text": "这是\n一段\n多行\n文本", "mode": "smart", "config": { "lineBreakReplacement": " ", "spaceHandling": "preserve", "trimEdges": true } }

响应示例

成功响应 (200):
{ "success": true, "data": { "originalText": "这是\n一段\n多行\n文本", "convertedText": "这是一段多行文本", "stats": { "originalLength": 20, "convertedLength": 15, "linesRemoved": 3, "wordsCount": 5 } } }
GET /api/v1/health 健康检查
检查API服务状态

响应示例

{ "status": "ok", "version": "2.3.0", "timestamp": "2025-09-05T12:00:00Z", "message": "服务运行正常" }
POST /api/v1/fetch-url 获取网页内容
获取网页内容并转换为Markdown格式

请求参数

参数 类型 必填 说明
url string 是 要获取的网页URL

请求示例

{ "url": "https://example.com/article" }

响应示例

{ "success": true, "data": { "content": "# 文章标题\n\n这是文章内容...", "title": "文章标题", "url": "https://example.com/article", "wordCount": 150 } }
GET /api/v1/stats 使用统计
获取工具使用统计信息

响应示例

{ "totalConversions": 1234, "popularModes": ["smart", "custom", "simple"], "averageTextLength": 500, "mostUsedFeatures": ["url_fetch", "smart_mode"] }
⚙️ 配置参数

当使用custom模式时，可以配置以下参数：

参数 类型 默认值 说明
lineBreakReplacement string " " 换行符替换字符
spaceHandling string "preserve" 空格处理方式：preserve, normalize, remove
trimEdges boolean true 是否去除首尾空格
maxLineLength number 0 最大行长（0表示无限制）
💡 提示： 所有API调用都是纯前端实现，无需网络请求，响应速度极快。
🎯 使用示例
// 完整使用示例 const api = window.apiClient; // 简单转换 const result1 = await api.convertText('Hello\nWorld', 'simple'); console.log(result1.convertedText); // "Hello World" // 智能转换 const result2 = await api.convertText('复杂\n多行\n文本', 'smart'); console.log(result2.convertedText); // "复杂多行文本" // 自定义转换 const result3 = await api.convertText('测试\n文本', 'custom', { lineBreakReplacement: ' → ', spaceHandling: 'normalize' }); console.log(result3.convertedText); // "测试 → 文本"
